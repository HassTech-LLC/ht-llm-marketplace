# Universal Examples

These examples are intentionally small so they can be copied into existing projects.

## Runnable Starters

- `node-terminal`: OpenAI-compatible terminal call using built-in `fetch`.
- `python`: OpenAI-compatible terminal call using the Python standard library.
- `plain-html`: Framework-neutral Web Component embed.
